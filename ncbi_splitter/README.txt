This program takes ncbi gene database as input ,fetches and prints the following details.
(Update when modified)
1.Official symbol
2.Official Name
3.Gene id
4.Gene type
5.Organism name
6.Organism Lineage
7.Summary
8.Alias list

The objective is to split the ncbi_gene file into pieces to facilitate faster xml parsing.
The functions can be used and modified to suit specific purposes, and new fetching
functions can be added with different xpath expressions,depending on the applications.
